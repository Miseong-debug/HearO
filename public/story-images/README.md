```
    - stage3.png  (중반: 동굴 탐험)
    - stage4.png  (후반: 드래곤 둥지)
    - stage5.png  (클라이맥스: 보스 전투)
    - stage6.png  (엔딩: 승리와 귀환)

  /zombie/
    - stage1.png  (시작: 좀비 아포칼립스 발생)

    - stage3.png  (중반: 생존자 캠프)
    - stage4.png  (후반: 물자 수집)
    - stage5.png  (클라이맥스: 좀비 호드 격퇴)
    - stage6.png  (엔딩: 안전지대 도착)

```

## 권장 이미지 사양

- 해상도: 1024x576 (16:9) 또는 1024x1024 (1:1)
- 포맷: PNG 또는 WebP
- 스타일: 테마에 맞는 일러스트 스타일
